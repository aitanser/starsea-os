核心功能特性

量子终端

80+ 内置命令，Tab 补全，历史记录，别名，收藏夹，多会话管理

虚拟文件系统

支持目录创建、文件读写、权限控制、软链接，数据 JSON 存储

定时任务

自动备份（10分钟）、定时报告（1小时）、Cron 表达式、自愈守护

颜色主题

7 种 ANSI 终端配色，深色/浅色模式一键切换

插件系统

动态加载 Python 模块，扩展终端功能，支持自定义命令

安全审计

所有敏感操作独立记录，支持检索和开发者审计

网络工具

ping、traceroute、SCP 安全传输、邮件通知

双因素认证

TOTP 动态码保护高权限账户，可选启用

安装与运行

v1.0.0

环境要求：Python 3.8+

依赖安装：

pip install -r requirements.txt

启动系统：

cd 组件 && python main.py
