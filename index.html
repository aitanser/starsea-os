核心功能特性

量子终端

80+ 内置命令，Tab 补全，历史记录，别名，收藏夹，多会话隔离。

虚拟文件系统

独立于真实磁盘，支持目录、文件、权限控制、软链接，永久化存储。

定时任务与守护

多色彩主题

插件系统 + 市场 NEW

应用商店 v1.2.0

`app` 命令安装官方应用（文件管理器、计算器、词典等），一键注册为终端命令。

安全审计与双因素

审计日志 + TOTP 动态验证，可选硬件绑定许可证。

网络工具集

独立 SDK

提供 starsea_sdk 包，外部程序可调用用户管理、VFS、加密接口。

系统架构

星海量子操作系统/
├── 组件/                     # 运行根目录
│   ├── main.py               # 统一启动器 (7种模式 CLI/UI/Web)
│   ├── install_wizard.py     # 图形化/静默安装向导
│   ├── requirements.txt
│   ├── system/
│   │   ├── config.py         # 多语言配置，密钥哈希
│   │   ├── utils.py          # 加密/解密，用户管理
│   │   ├── terminal.py       # 终端核心 (pm, app, help 分类)
│   │   ├── plugin_market.py  # 插件市场
│   │   ├── app_market.py     # 应用市场
│   │   ├── web_server.py     # Flask Web 面板 (0.0.0.0)
│   │   ├── gui_launcher.py   # PyQt5/Tkinter 图形面板
│   │   └── ... (40+ 模块)
│   ├── 关于/  数据集/  示例文档/
│   └── 应用/  插件/           # 安装目录
└── G_D_*_数据储存库/          # 用户数据仓库

快速安装

1
环境要求
Python 3.8+
2
安装依赖
pip install -r requirements.txt
3
图形化安装向导 (推荐)
python install_wizard.py
4
启动系统
python main.py # 选择 1/2/3 进入不同模式
💡 静默安装: python install_wizard.py --unattended --product-key ...
