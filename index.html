快速安装

1
环境要求
Python 3.8+
2
安装依赖
pip install -r requirements.txt
3
图形化安装向导 (推荐)
python install_wizard.py
4
启动系统
python main.py # 选择 1/2/3 进入不同模式
💡 静默安装: python install_wizard.py --unattended --product-key ...

版本规范

📌 版本号格式

<主>.<次>.<修订>[-<阶段>.<序号>]

主版本号：不兼容变更
次版本号：新增功能（向下兼容）
修订号：问题修复
阶段：alpha / beta / rc

🔄 递进规则

修复Bug修订号 +1
新增功能次版本号 +1，修订归零
不兼容变更主版本号 +1，次修归零
beta → rc → 正式阶段变更，序号重置

📦 打包策略

alpha❌ 仅源码
beta / 正式版✅ 必须打包
rc❌ 源码验证

🔑 产品密钥绑定

每个打包发布版本（beta 和正式版）生成唯一密钥和 bcrypt 哈希。
alpha / rc 沿用上一版密钥。
