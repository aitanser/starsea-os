**版本：1.0.0-beta.1**  
**发布日期：2025-06-25**

▼ 新增功能  

**一、安全增强**  
- 系统产品密钥采用 bcrypt 哈希存储，支持暴力攻击锁定（5次错误锁定15分钟）  
- 用户密码 SHA-256 加盐哈希  
- 双因素认证（TOTP）支持（可选）  
- 硬件绑定许可证验证  
- 开发者独立激活密钥机制  
- 安全审计日志记录所有敏感操作  
- 配置文件加解密工具（支持终端内安全阅读加密内容）  

**二、终端体验**  
- 超过 80 个内置命令（系统管理、文件操作、网络诊断、资源监控、虚拟文件系统等）  
- Tab 命令自动补全、历史记录、别名与收藏夹  
- 7 种 ANSI 颜色主题  
- 实时时钟显示与消息广播  
- 性能分析命令（`time`）、沙盒执行环境  

**三、多会话与虚拟文件系统**  
- 多会话管理，每个会话拥有独立的虚拟文件系统和工作目录  
- 虚拟文件系统支持目录创建、文件读写、权限控制、软链接  

**四、运维管理**  
- 系统维护工具（完整性检查、临时文件清理、缺失文件修复）  
- 还原点管理（创建/恢复快照）、版本化备份  
- 进程查看与任务管理、磁盘使用统计  
- 系统设置菜单（可修改锁定策略）  
- 自愈守护进程（监控调度器、自动压缩日志）  

**五、扩展与集成**  
- 插件系统（动态加载 `plugins/` 目录下的 Python 模块）  
- 星海脚本引擎（支持 `.ssea` 脚本执行）  
- Cron 表达式定时任务、GitHub Release 自动更新  
- 网络工具（ping、traceroute、SCP 文件传输）、邮件通知、用户组管理  

**六、数据与文档**  
- 内置 8 种语言完整界面翻译  
- 数据集（ASCII 码表、Unicode 区块、150+ 表情集、示例表格、测试文本）  
- 16 类空白模板文件（涵盖文本、图片、音频、视频、数据库、代码等）  
- 自动生成安装包脚本

**版本：1.0.0**  
**发布日期：2026-05-13**

▼ 新增功能  

**一、独立 SDK（starsea-os-sdk）**  
- 独立 Python 包，可脱离系统单独安装和使用  
- 提供 `StarseaClient` 类，封装配置文件加密读写、用户管理、虚拟文件系统完整操作  
- 支持基于产品密钥或 Fernet 派生密钥初始化  
- 提供专用异常类（StarseaError、AuthenticationError、ConfigError）  
- 内置 SDK 检测脚本，一键验证全部 API 可用性  

**二、官方应用**  
- 文件管理器、应用商店、图库、文本编辑器、系统监视器、日历与提醒、备份还原向导、科学计算器、星海便签、星海词典（共10个）  

**三、官方插件**  
- 随机密码生成器、增强系统信息、随机笑话、名言警句  

**四、自动化测试套件**  
- 基于 pytest 的完整测试框架，覆盖核心模块  

**五、系统改进**  
- 修复 Windows 下 readline 依赖问题（改用 pyreadline3）  
- 数据储存库自动重命名逻辑  
- 日常检测报告仅显示已初始化语言  
- 会话管理增强、SDK 路径自动注册  
- 完善安装包生成脚本，统一代码注释风格

**版本：1.2.0**  
**发布日期：2026-06-15**

▼ 新增功能  
- **市场生态完善**：统一市场源 `market_config.json`，新增 `pm list-remote` / `app list-remote` 命令，抽取 `market_common.py` 公共模块  
- **终端体验优化**：`help` 命令改版（按类别显示、关键词搜索），命令补全支持 `pm`/`app` 所有子命令，`vwrite` 支持交互式多行输入  
- **图形界面现代化**：`gui_launcher.py` 重构（PyQt5 优先，自动回退 Tkinter），9 个标签页，多语言支持  
- **安装向导升级**：`install_wizard.py` 图形化分步引导，支持静默安装，自动备份与回滚  
- **统一日志系统**：`system/unified_logger.py`（JSON Lines + 审计日志），日志级别控制  
- **配置文件路径统一**：所有 `.json` 配置移至 `config/` 目录，`get_config_path()` 统一获取  
- **多平台内核增强**：新增 Android 支持（设备型号、电池、应用列表、WiFi）；增强 Linux/Windows/macOS 内核信息（GPU、USB、蓝牙、服务等）  
- **启动器多模式与零硬编码**：`main.py` 支持 7 种启动模式，通过环境变量传递密钥  
- **Web 面板独立化**：新增市场 API，支持 `pm`/`app` 命令，监听 `0.0.0.0`，HTTP Basic Auth  
- **完整的单元测试套件**：新增多个测试文件，提供 `run_tests.py` 一键运行  

▼ 修复  
- 修复 Windows 下 `getpass` 警告  
- 修复 `login_handler.py` 调用 `star_sea_terminal` 参数错误  
- 修复 Web 面板 `'NoneType' object has no attribute 'vfs_list'`  
- 修复 `app list-remote` 网络超时导致缓存失效  
- 修复 `install_wizard.py` 中 `hash_data` 未定义  
- 修复 `gui_launcher.py` Tkinter 回退版 VFS 空指针  
- 修复 `windows_kernel.py` 语法错误  
- 修复 `terminal.py` 中 `CURRENT_VERSION` 和 `ROLE_DEVELOPER` 未导入  
- 修复 `utils.py` 中 `log_action` 重复定义  
- 修复市场缓存 JSON 编码问题  
- 修复配置路径迁移时导致的循环导入  

▼ 优化  
- 所有文件读写明确指定 `encoding='utf-8'`  
- 统一异常处理，添加友好提示  
- 代码风格统一，使用 `#` 注释  
- 精简 `setup_package.py`，优化市场下载重试机制
